# Images

This directory contains app images and graphics for the Drachma Mobile Wallet.

## Structure

- **logos/** - App logos and branding
- **backgrounds/** - Background images for screens
- **illustrations/** - Illustrations for onboarding and empty states
- **splash/** - Splash screen assets

## Guidelines

- Use PNG format for images with transparency
- Use JPG format for photographs
- Optimize all images for mobile (max 2x resolution)
- Use WebP format when supported for better compression
- Keep file sizes minimal for app performance

## Naming Convention

- Use lowercase with hyphens: `logo-primary.png`
- Include resolution suffix when needed: `background-home@2x.png`
- Use descriptive names: `illustration-empty-wallet.png`
